### 🧠 Thinking Skills to Build This Semester

#### Learn These 4 Mental Models:
1. **Feynman Technique** — Explain any concept in simple words. If you can't, you don't understand it yet.
2. **Spaced Repetition** — Use Anki flashcards for formulas, concepts, and definitions. Review daily.
3. **Active Recall** — After every lecture, close it and write what you remember. Never passive re-watching.
4. **Pomodoro Method** — 25 min deep work → 5 min break. Protect your focus fiercely.

#### Math Anxiety Protocol:
- Do NOT skip math. Do NOT fear it.
- Rule: If you don't understand something, **draw it** before you Google it.
- Spend 20 min/day on Khan Academy math — daily, no exceptions, this month only.
- Goal by end of Semester 0: Comfortable with fractions, exponents, basic algebra, and reading a simple graph.

## 🔬 Research & Publication System

> **Philosophy:** You don't need a PhD to publish. You need a good question, rigorous methodology, and persistence.

### Phase 1 — Consumer (Semesters 1–3)
- Read 1–2 papers per week from ArXiv, PapersWithCode, Distill.pub
- Use [Connected Papers](https://connectedpapers.com) to map research areas
- Annotate using [Hypothesis](https://web.hypothes.is) or Obsidian notes
- Write **"Paper in Plain English"** summaries on LinkedIn and Medium

### Phase 2 — Reproducer (Semesters 4–5)
- Pick a published paper with open-source code
- Reproduce results on a new dataset
- Document differences + new findings
- Publish reproduction as a GitHub repo + blog post
- Submit to [Papers With Code](https://paperswithcode.com)

### Phase 3 — Contributor (Semesters 6–7)
- Find a **gap** in existing research (look for "limitations" sections)
- Formulate a research question
- Run experiments with proper controls
- Write using the IMRaD structure: Introduction, Methods, Results, Discussion
- Tools: Overleaf (LaTeX), Zotero (citations), Matplotlib (plots)

### Phase 4 — Publisher (Semester 8)
- Submit to workshop tracks first (lower bar, faster feedback):
  - ICLR Tiny Papers
  - NeurIPS Workshop
  - EMNLP Student Research Workshop
- Iterate on reviewer feedback
- Target full conference papers (ACL, NeurIPS, ICML) by Year 4
