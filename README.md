# 🎓 NeuroNova: The Open AI/Data Science University

A self-paced, 8-semester Bachelor-level learning roadmap using **only free YouTube university playlists** (MIT, Stanford, Harvard), certifications, GitHub projects, LinkedIn content, LeetCode problems, hackathons, and remote internships.

💼 Goal: Become **job-eligible in 5 months**, and fully industry-ready in 2.5 years.

---

## 📅 Weekly Commitment
- 🕐 **Weekdays**: 2–4 hrs/day
- 🧠 **Weekends**: 6–7 hrs/day

---

## 🧩 Structure of Each Semester
- 🎥 **University Playlist**
- 🧪 **Certifications**
- 💻 **GitHub Project**
- 📝 **LinkedIn Post Ideas**
- 🧠 **LeetCode Problems**
- 🏁 **Competitions / Hackathons**
- 🌐 **Online Extracurriculars**
- 🧑‍💼 **Remote Internships**
